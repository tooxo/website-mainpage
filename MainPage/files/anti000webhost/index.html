Anti-000Webhost


This is a JavaScript + CSS combination, which hides the overlay, put in place by the free program of 000webhost.com, a common free hoster.


Seems cool, but how do I get it?


First you need to get the files. You can download them for yourself and add them to your server or simply include them by url.
Server1: s.chulte.de [FILE1-COPY] [FILE2-COPY]
Server2: github.com [FILE1-COPY] [FILE2-COPY]


Now I got the files, what to do with them?


Well, now you only need to implement them to every single visible web-page hosted on 000WebHost. Then it should work.


Wait, how do I implement them?


Okay, okay, okay, it's very easy. You only need to add these both lines to every HTML document used by you.
Add this: [MIRROR1-COPY] [MIRROR2-COPY] anywhere in the file.
Then add this: [MIRROR1-COPY] [MIRROR1-COPY] in the header of the file.


And now, you are ready.
