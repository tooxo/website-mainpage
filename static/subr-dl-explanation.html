How does it work?

It takes the given amount of Images from a given subreddit
and bundles it into a zip-Archive for you ready to download.
This might be useful, if you need a bunch of specific images
of a specific kind.
Usage instructions:
put the name of the subreddit in the first field, no url,
no r/ something, the subreddit name only.
Then put the amount of images to be pulled in the second
input field. Press Submit and wait for the result.
